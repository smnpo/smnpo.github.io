## 简介
### SMNPO is short for Software Melon Nonlinear Pipeline Organization.
### 软件瓜非线性流水线组织 简称 软件瓜。
* 别问我为啥叫软件瓜非线性流水线组织，我自己也布吉，这就是一种缘分。软件瓜的瓜是傻瓜的瓜，我们软件也希望都已各种瓜来命名。至于我们要做些啥，后面用交付内容来体现。
